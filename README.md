## C# Projects

This repository contains small projects developed using C# and the .NET Framework 
as part of my software development training.

## Projects

### Car Insurance 

An ASP.NET MVC web application that simulates a car insurance quote system.

**Features:**
- User input for insurance policies
- Quote calculation using business logic
- Admin page to view issued quotes

**Technologies:**
- C#
- ASP.NET MVC
- Entity Framework
- SQL Server

### NFL Stats

A data visualization web application designed to store and display NFL football 
statistics for fantasy football purposes.

**Features:**
- Stores and displays football statistics
- Uses Entity Framework with Code First workflow

**Technologies:**
- C#
- ASP.NET MVC
- Entity Framework 6
- SQL Server

> This project is still a work in progress.
